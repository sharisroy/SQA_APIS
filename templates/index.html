Welcome to SQA API Documentation

This documentation covers all endpoints for Authentication and Note Management. Use the sidebar to navigate through API sections.

All APIs require Bearer Token in Authorization header (except signup and login). The API follows REST principles and returns JSON responses.

Secure Authentication

JWT-based authentication with access tokens

Note Management

Full CRUD operations for personal notes

RESTful API

Clean, consistent API design following REST principles

Signup

POST /signup

Register a new user account.

Request Body 
{
  "name": "John Doe",
  "email": "john@example.com",
  "password": "password123"
}
Response Body 
{
  "code": 200,
  "message": "User registered successfully",
  "user": {
    "name": "John Doe",
    "email": "john@example.com"
  }
}

Login

POST /login

Authenticate user and receive access token.

Request Body 
{
  "email": "john@example.com",
  "password": "password123"
}
Response Body 
{
  "code": 200,
  "message": "Login successful",
  "user": {
    "name": "John Doe",
    "email": "john@example.com"
  },
  "auth": {
    "access_token": "eyJhbGciOiJI...",
    "expires_at": "2025-10-10T14:00:00+06:00"
  }
}

Profile

GET /profile/me

Get current user's profile information.

Headers 
Authorization: Bearer <access_token>
Response Body 
{
  "code": 200,
  "message": "User profile retrieved successfully",
  "profile": {
    "name": "John Doe",
    "email": "john@example.com",
    "id": "646c8190-378a-404d-98f0-0c9e666a7eea"
  }
}

Create Note

POST /note

Create a new note.

Headers 
Authorization: Bearer <access_token>
Request Body 
{
  "title": "Shopping List",
  "note": "Buy groceries for the week"
}
Response Body 
{
  "code": 201,
  "message": "Note created successfully",
  "note": {
    "id": "3c34a6d6-ab4e-485a-9ca3-c292aef8a1df",
    "title": "Shopping List",
    "note": "Buy groceries for the week"
  }
}

Get Note by ID

GET /note/{id}

Retrieve a specific note by its ID.

Headers 
Authorization: Bearer <access_token>
Response Body 
{
  "code": 200,
  "message": "Note retrieved successfully",
  "note": {
    "id": "3c34a6d6-ab4e-485a-9ca3-c292aef8a1df",
    "title": "Shopping List",
    "note": "Buy groceries for the week",
    "created_at": "2025-09-18T20:38:48.030888+00:00",
    "updated_at": null,
    "user_id": "646c8190-378a-404d-98f0-0c9e666a7eea"
  }
}

Update Note

PATCH /note/{id}

Update an existing note.

Headers 
Authorization: Bearer <access_token>
Request Body 
{
  "title": "Updated Shopping List",
  "note": "Buy groceries and milk"
}
Response Body 
{
  "code": 200,
  "message": "Note updated successfully",
  "note": {
    "id": "3c34a6d6-ab4e-485a-9ca3-c292aef8a1df",
    "title": "Updated Shopping List",
    "note": "Buy groceries and milk",
    "created_at": "2025-09-18T20:38:48.030888+00:00",
    "updated_at": "2025-10-09T05:00:00+00:00",
    "user_id": "646c8190-378a-404d-98f0-0c9e666a7eea"
  }
}

Delete Note

DELETE /note/{id}

Delete a note by its ID.

Headers 
Authorization: Bearer <access_token>
Response Body 
{
  "code": 200,
  "message": "Note deleted successfully"
}
