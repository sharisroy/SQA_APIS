Welcome to SQA API

This project is designed for practicing API testing, including both manual and automated testing with Postman.

Topics Covered:

  • REST API Concepts: Endpoints, methods, headers, query params, status codes.
  • User Authentication: Signup, login, token-based access, expired/invalid token handling.
  • Profile Management: CRUD operations, response validation, schema checks.
  • Notes API: CRUD notes, pagination, required field validation.
  • Response Validation: Status codes, JSON schema, error messages.
  • Positive & Negative Testing: Valid and invalid input scenarios.
  • SQA Perspectives: Maintainability, reliability, performance, security checks.
  • Automation Testing: Postman collections, environment variables, scripts, test suites.
  • Reporting & Debugging: Logging, Postman console, Supabase logs.

Learning Outcomes:

  • Test APIs manually and validate scenarios.
  • Understand SQA best practices for API quality assurance.
  • Write automated API test scripts using Postman.
  • Validate responses, perform security and performance testing.

Signup API

Endpoint: POST auth/signup

Request Body:

{
    "name": "John Doe",
    "email": "john@example.com",
    "password": "your_password"
}

Response Example:

{
    "success": true,
    "code": 200,
    "message": "User registered successfully.",
    "user": {
        "email": "john@example.com",
        "name": "John Doe"
    }
}

Tests Script:

pm.test("Status 200", () => pm.response.to.have.status(200));
pm.test("Success true", () => pm.expect(pm.response.json().success).to.eql(true));

Login API

Endpoint: POST auth/login

Request Body:

{
    "email": "john@example.com",
    "password": "your_password"
}

Response Example:

{
    "success": true,
    "code": 200,
    "message": "Login successful.",
    "user": { "email": "john@example.com", "name": "John Doe" },
    "auth": { "access_token": "xxxxxx", "expires_at": 1690000000 }
}

Tests Script:

var jsonData = pm.response.json();
pm.test("Login success", () => pm.expect(jsonData.success).to.eql(true));
if(jsonData.auth && jsonData.auth.access_token){
    pm.environment.set("access_token", jsonData.auth.access_token);
}

Profile API

Endpoint: GET profile/me

Headers:

Authorization: Bearer <access_token>

Response Example:

{
    "success": true,
    "code": 200,
    "message": "Profile retrieved successfully.",
    "user": { "email": "john@example.com", "name": "John Doe" }
}

Tests Script:

var jsonData = pm.response.json();
pm.test("Status 200", () => pm.response.to.have.status(200));
pm.test("User email present", () => pm.expect(jsonData.user.email).to.exist);

Notes APIs

Add Note

Endpoint: POST /note

Headers:

Authorization: Bearer <access_token>

Request Body:

{
    "title": "Meeting Notes",
    "note": "Project kickoff scheduled for Monday."
}

Response Example:

{
    "success": true,
    "code": 200,
    "note": { "id": "uuid-generated", "title": "Meeting Notes" }
}

Tests Script:

var jsonData = pm.response.json();
pm.test("Note added", () => pm.expect(jsonData.success).to.eql(true));
if(jsonData.note && jsonData.note.id){
    pm.environment.set("note_id", jsonData.note.id);
}

Get Note by ID

Endpoint: GET /note/<note_id>

Headers:

Authorization: Bearer <access_token>

Response Example:

{
    "success": true,
    "code": 200,
    "note": {
        "id": "uuid-generated",
        "title": "Meeting Notes",
        "note": "Project kickoff scheduled for Monday."
    }
}

Tests Script:

var jsonData = pm.response.json();
pm.test("Note retrieved", () => pm.expect(jsonData.success).to.eql(true));
pm.test("Note title present", () => pm.expect(jsonData.note.title).to.exist);

Get All Notes (with pagination)

Endpoint: GET /note?page=1&per_page=10

Headers:

Authorization: Bearer <access_token>

Response Example:

{
    "success": true,
    "code": 200,
    "page": 1,
    "per_page": 10,
    "total_notes": 5,
    "notes": [
        { "id": "uuid1", "title": "Meeting Notes" },
        { "id": "uuid2", "title": "Shopping List" }
    ]
}

Tests Script:

var jsonData = pm.response.json();
pm.test("Success true", () => pm.expect(jsonData.success).to.eql(true));
pm.test("Notes array exists", () => pm.expect(jsonData.notes).to.be.an("array"));

Update Note

Endpoint: PATCH /note/<note_id>

Headers:

Authorization: Bearer <access_token>

Request Body:

{
    "title": "Updated Meeting Notes",
    "note": "Rescheduled to Tuesday at 3 PM."
}

Response Example:

{
    "success": true,
    "code": 200,
    "note": {
        "id": "uuid-generated",
        "title": "Updated Meeting Notes",
        "note": "Rescheduled to Tuesday at 3 PM."
    }
}

Tests Script:

var jsonData = pm.response.json();
pm.test("Note updated", () => pm.expect(jsonData.success).to.eql(true));
pm.test("Updated title matches", () => pm.expect(jsonData.note.title).to.eql("Updated Meeting Notes"));
pm.test("Updated note matches", () => pm.expect(jsonData.note.note).to.eql("Rescheduled to Tuesday at 3 PM."));
