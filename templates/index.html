Welcome to Your API Project

This project helps you practice both manual and automated API testing using Postman.

API Endpoints

  • Signup: POST /signup

    Request Body (JSON):

    {
    "name": "John Doe",
    "email": "john@example.com",
    "password": "your_password"
}
  • Login: POST /login

    Request Body (JSON):

    {
    "email": "john@example.com",
    "password": "your_password"
}
  • Profile: GET /me

    Headers:

    Authorization: Bearer <access_token>

Steps to Test in Postman

  1. Open Postman and create a new request.
  2. For Signup/Login, select POST and add JSON body as shown above.
  3. Click Send. You should receive a JSON response with your user info and access token.
  4. For Profile, select GET, add Authorization header: Bearer <access_token>, then click Send.

Tips

  • Keep your access_token safe; it authenticates your requests.
  • All endpoints return JSON with success, code, and message.
  • Use Postman environment variables for access_token to simplify testing multiple requests.
