Welcome to SQA API

This project is designed for practicing API testing, including both manual and automated testing with Postman.

Topics Covered:

  • REST API Concepts: Endpoints, methods (GET, POST, PATCH, DELETE), headers, query params, status codes.
  • User Authentication: Signup, login, token-based access, expired/invalid token handling.
  • Profile Management: CRUD operations, response validation, schema checks.
  • Notes API: CRUD notes, pagination, validation for required fields.
  • Response Validation: Status codes, JSON schema, error messages.
  • Positive & Negative Testing: Valid and invalid input testing.
  • SQA Perspectives: Maintainability, reliability, performance, security checks, edge cases.
  • Automation Testing: Postman collections, environment variables, scripts, test suites.
  • Reporting & Debugging: Logging, debugging, Postman console, Supabase logs.

Learning Outcomes:

  • Test APIs manually and validate all scenarios.
  • Understand SQA best practices for API quality assurance.
  • Write automated API test scripts using Postman.
  • Validate responses, perform security and performance testing.

Signup API

Endpoint: POST auth/signup

Request Body (JSON):

{
    "name": "John Doe",
    "email": "john@example.com",
    "password": "your_password"
}

Response Example:

{
    "success": true,
    "code": 200,
    "message": "User registered successfully.",
    "user": {
        "email": "john@example.com",
        "name": "John Doe"
    }
}

Postman Instructions: Select POST, add JSON body, click Send.

Pre-request Script:

Tests Script:

// Validate response
pm.test("Status 200", () => pm.response.to.have.status(200));
pm.test("Success true", () => pm.expect(pm.response.json().success).to.eql(true));

Login API

Endpoint: POST auth/login

Request Body (JSON):

{
    "email": "john@example.com",
    "password": "your_password"
}

Response Example:

{
    "success": true,
    "code": 200,
    "message": "Login successful.",
    "user": { "email": "john@example.com", "name": "John Doe" },
    "auth": { "access_token": "xxxxxx", "expires_at": 1690000000 }
}

Postman Instructions: Select POST, add JSON body, click Send.

Pre-request Script:

Tests Script:

// Check success & store token
var jsonData = pm.response.json();
pm.test("Login success", () => pm.expect(jsonData.success).to.eql(true));
if(jsonData.auth && jsonData.auth.access_token){
    pm.environment.set("access_token", jsonData.auth.access_token);
}

Profile API

Endpoint: GET profile/me

Headers:

Authorization: Bearer <access_token>

Response Example:

{
    "success": true,
    "code": 200,
    "message": "Profile retrieved successfully.",
    "user": { "email": "john@example.com", "name": "John Doe" }
}

Postman Instructions: Select GET, add Authorization header, click Send.

Pre-request Script:

// No specific script needed, just ensure token is set

Tests Script:

// Validate profile response
var jsonData = pm.response.json();
pm.test("Status 200", () => pm.response.to.have.status(200));
pm.test("User email present", () => pm.expect(jsonData.user.email).to.exist);

Notes APIs

Add Note

Endpoint: POST /note

Headers:

Authorization: Bearer <access_token>

Request Body:

{
    "note": "Buy groceries"
}

Response Example:

{
    "success": true,
    "code": 200,
    "message": "Note added successfully.",
    "note_id": "uuid-generated"
}

Postman Instructions: Select POST, add Authorization & JSON body, click Send.

Pre-request Script:

Tests Script:

var jsonData = pm.response.json();
pm.test("Note added", () => pm.expect(jsonData.success).to.eql(true));
if(jsonData.note_id) pm.environment.set("note_id", jsonData.note_id);

Get Note by ID

Endpoint: GET /note/<note_id>

Headers:

Authorization: Bearer <access_token>

Response Example:

{
    "success": true,
    "code": 200,
    "note": { "id": "uuid-generated", "note": "Buy groceries" }
}

Tests Script:

var jsonData = pm.response.json();
pm.test("Note retrieved", () => pm.expect(jsonData.success).to.eql(true));
pm.test("Note id matches", () => pm.expect(jsonData.note.id).to.eql(pm.environment.get("note_id")));

Get All Notes

Endpoint: GET /note?page=1&per_page=10

Headers:

Authorization: Bearer <access_token>

Response Example:

{
    "success": true,
    "code": 200,
    "page": 1,
    "per_page": 10,
    "total_notes": 5,
    "notes": [
        { "id": "uuid1", "note": "Buy groceries" },
        { "id": "uuid2", "note": "Pay bills" }
    ]
}

Tests Script:

var jsonData = pm.response.json();
pm.test("Success true", () => pm.expect(jsonData.success).to.eql(true));
pm.test("Notes array exists", () => pm.expect(jsonData.notes).to.be.an("array"));
