Welcome to SQA API Project

This project helps you practice both manual and automated API testing using Postman.

Use the menu on the left to jump to specific API instructions.

Signup API

Endpoint: POST /signup

Request Body (JSON):

{
    "name": "John Doe",
    "email": "john@example.com",
    "password": "your_password"
}

Response Example:

{
    "success": true,
    "code": 200,
    "message": "User registered successfully.",
    "user": {
        "email": "john@example.com",
        "name": "John Doe"
    }
}

Postman Instructions: Select POST, add JSON body as above, click Send.

Login API

Endpoint: POST /login

Request Body (JSON):

{
    "email": "john@example.com",
    "password": "your_password"
}

Response Example:

{
    "success": true,
    "code": 200,
    "message": "Login successful.",
    "user": { "email": "john@example.com", "name": "John Doe" },
    "auth": { "access_token": "xxxxxx", "expires_at": 1690000000 }
}

Postman Instructions: Select POST, add JSON body, click Send.

Profile API

Endpoint: GET /me

Headers:

Authorization: Bearer <access_token>

Response Example:

{
    "success": true,
    "code": 200,
    "message": "Profile retrieved successfully.",
    "user": { "email": "john@example.com", "name": "John Doe" }
}

Postman Instructions: Select GET, add Authorization header, click Send.

Notes APIs

Add Note

Endpoint: POST /note

Headers:

Authorization: Bearer <access_token>

Request Body:

{
    "note": "Buy groceries"
}

Response Example:

{
    "success": true,
    "code": 200,
    "message": "Note added successfully.",
    "note_id": "uuid-generated"
}

Get Note by ID

Endpoint: GET /note/<note_id>

Headers:

Authorization: Bearer <access_token>

Response Example:

{
    "success": true,
    "code": 200,
    "note": { "id": "uuid-generated", "note": "Buy groceries" }
}

Get All Notes

Endpoint: GET /note?page=1&per_page=10

Headers:

Authorization: Bearer <access_token>

Response Example:

{
    "success": true,
    "code": 200,
    "page": 1,
    "per_page": 10,
    "total_notes": 5,
    "notes": [
        { "id": "uuid1", "note": "Buy groceries" },
        { "id": "uuid2", "note": "Pay bills" }
    ]
}

Update Note

Endpoint: PATCH /note/<note_id>

Headers:

Authorization: Bearer <access_token>

Request Body:

{
    "note": "Updated note text"
}

Delete Note

Endpoint: DELETE /note/<note_id>

Headers:

Authorization: Bearer <access_token>
